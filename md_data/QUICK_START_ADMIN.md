## What You Can Do

### Create New Teacher
1. Go to Teachers page
2. Click "Add New Teacher"
3. Fill in details
4. Click "Add Teacher"

### Create New Student
1. Go to Students page
2. Click "Add New Student"
3. Fill in details
4. Click "Add Student"

### Create New Class
1. Go to Classes page
2. Click "Add New Class"
3. Fill in details
4. Click "Create Class"

### Create New Subject
1. Go to Subjects page
2. Click "Add New Subject"
3. Fill in details
4. Click "Create Subject"

### Create New Fee
1. Go to Fees page
2. Click "Add New Fee"
3. Fill in details
4. Click "Create Fee"

### Create New Bus Route
1. Go to Transport page
2. Click "Add Route"
3. Fill in route details
4. Click "Create Route"

### Register New Bus
1. Go to Transport page
2. Click "Add Bus"
3. Fill in bus details
4. Click "Register Bus"

### Create Monthly Exam
1. Go to Results page
2. Select Class
3. Select Subject
4. Click "Create Exam"
5. Enter marks for students
6. Click "Upload Marks"

## If Data Doesn't Show

1. **Refresh Page**: Ctrl+R (Windows) or Cmd+R (Mac)
2. **Clear Cache**: Ctrl+Shift+Delete (Windows) or Cmd+Shift+Delete (Mac)
3. **Clear LocalStorage**: DevTools → Application → Local Storage → Clear All
4. **Check Console**: F12 → Console for errors

## Need Help?

- Check browser console (F12) for error messages
- Verify backend is running on port 5001
- Verify JWT token is valid
- Check collegeId in localStorage
