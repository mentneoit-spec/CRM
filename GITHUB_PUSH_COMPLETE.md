## 🚀 Next Steps

### For Team Members:

1. **Clone the Repository:**
   ```bash
   git clone https://github.com/UVineethaa/collegedata.git
   cd collegedata
   ```

2. **Setup Backend:**
   ```bash
   cd backend
   npm install
   cp .env.example .env
   # Update .env with your database credentials
   npm start
   ```

3. **Setup Frontend:**
   ```bash
   cd frontend
   npm install
   npm start
   ```

4. **Create Test Data:**
   ```bash
   cd backend
   node setup-test-college.js
   node add-test-data.js
   ```
