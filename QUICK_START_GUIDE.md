2. Select: Rohan Singh
3. Select: Biology

## Quick Test - Import Students

1. Click pink **"Import Students CSV"** button
2. Click "Download CSV Template"
3. Open in Excel, add your students
4. Save the file
5. Click "Select CSV File"
6. Choose your file
7. Click "Upload & Import Students"
8. ✅ See results!

---

## Sample CSV Format

## What Each Feature Does

### Send Marks Email
✅ Enter student marks
✅ Automatically calculates percentage & grade
✅ Sends beautiful HTML email
✅ Saves to database
✅ Shows success confirmation

### Import Students CSV
✅ Upload CSV file
✅ Auto-creates classes & sections
✅ Auto-generates emails
✅ Shows created students with passwords
✅ Reports any errors
✅ Bulk import in seconds
