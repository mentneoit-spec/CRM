## 🔧 Troubleshooting

| Issue | Solution |
|-------|----------|
| Data not visible | Refresh page (F5) or click Refresh button |
| "Class not found" | Run import script first to create classes |
| "Email already exists" | Use "Update if exists" mode |
| Import too slow | Wait for completion, don't close browser |
| Login fails | Check email and password are correct |
