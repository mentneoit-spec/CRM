## 🚀 HOW TO LOGIN

1. Go to `http://localhost:3000`
2. Click **"Login"**
3. Select **Role**: Admin / Teacher / Student / Parent
4. Enter **Email** and **Password** from above
5. Click **Sign In**

---

## ✅ AFTER LOGIN - ACCESS AI FEATURES

Once logged in, navigate to:

- **Admin**: `http://localhost:3000/admin/ai`
- **Teacher**: `http://localhost:3000/teacher/ai`
- **Student**: `http://localhost:3000/student/ai`
- **Parent**: `http://localhost:3000/parent/ai`

---

## ⚠️ IF YOU SEE "NETWORK ERROR":

Make sure **Backend Server** is running:

```bash
cd backend
npm run dev
```

Or if using Python:
```bash
python manage.py runserver
```

Check that backend is running on port **5000** or **8000**
